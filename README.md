• Retrieval-Augmented Generation (RAG) for academic queries

## Hybrid RAG Chatbot
• Intelligent chatbot using Retrieval-Augmented Generation  
• Combines vector database search with LLM responses  
• Provides context-aware answers from document knowledge bases  
• Designed for scalable AI-powered information retrieval  

## Security Guard Monitoring System
• Computer Vision system to ensure guards remain alert during duty  
• Uses webcam monitoring with OpenCV face detection  
• Periodic image capture with automated logging system

• Controls volume, play/pause, and navigation using gestures  
• Real-time webcam-based interaction system
